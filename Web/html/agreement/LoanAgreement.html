2. 协议的订立及支付

2.1 各方同意并确认，各方通过自行或授权有关方根据左右逢园网站有关规则和说明，在左右逢园网站进行借款申请和投标操作等方式确认签署本协议。

2.2 各方通过上述方式接受本协议且丙方审核通过时，本协议立即成立；协议成立的同时甲方不可撤销地授权丙方将出借款项在扣除相关费用后划转、支付给乙方，出借款项由丙方根据授权划转完毕时本协议正式生效。

3. 各方权利和义务

3.1 甲方的权利和义务

3.1.1 甲方保证其所用于出借的资金来源合法，甲方是该资金的合法所有人，如果第三人对资金归属、合法性问题发生争议，由甲方负责解决。如甲方未能解决，则放弃享有其所出借款项所带来的利息收益。

3.1.2 甲方享有其所出借款项所带来的利息收益，并应主动缴纳由利息所得带来的可能的税费。

3.1.3 就甲方在本协议项下对乙方的部分或全部债权，甲方有权根据自己的意愿对外转让，并授权丙方可以视情况（必要时）将债权转让事项通知乙方和丁方。

3.1.4 如乙方违约，甲方有权要求丙方提供其已获得的乙方及丁方的信息。

3.1.5 甲方承诺对依据本协议获得的乙方信息或资料予以保密，除用于本协议目的进行出借与合理催收外，不得向外转让或披露。

3.2 乙方权利和义务

3.2.1 乙方必须按期足额向甲方归还每期应还本金和利息，向丙方按期足额支付相关借款管理费和服务费用，并支付与乙方逾期及提前还款有关的费用。

3.2.2 乙方承诺所借款项不用于任何违法用途。

3.2.3 乙方确认甲方可以根据自己的意愿对其根据本协议形成的对乙方的部分或全部债权进行转让，并特此同意甲方届时转让债权无需另行通知乙方；同时乙方承诺若届时需要，将采取必要措施配合债权的一次或多次转让或届时债权人债权的实现。在债权转让后，甲方在本协议项下的权利和义务自动转移到债权受让人名下，乙方应按照本协议的约定向债权受让人支付每期应还借款本息，不得以未接到债权转让通知为由拒绝履行还款义务。

3.2.4 乙方应确保其提供的信息和资料的真实性，不得提供虚假信息或隐瞒重要事实。若乙方违反本协议约定，丙方有权根据本协议及有关条款对乙方的信息和资料予以合理的披露。

3.2.5 乙方同意，甲方和丙方有权使用其自行收集或乙方提供的乙方资料和信息用于以下用途（包括但不限于）：

（1）为了提供本协议项下的服务，丙方向有关的合作机构提供必要之资料；

（2）用于解决争议、对纠纷进行调停等。

3.2.6 乙方有权了解其在丙方的信用评审进度及结果。

3.2.7 乙方不得将本协议项下的任何权利义务转让给任何其他方。

3.2.8 乙方同意并确认，本协议如涉及两人以上借款，任一借款人均应履行本合同项下的义务，对全部借款承担连带清偿责任，甲方和丙方有权向任一借款人追索本息及其他相关费用。

3.3 丙方的权利和义务

3.3.1 丙方有权根据乙方提供的各项信息及丙方独立获得的信息评定乙方在丙方处所拥有的个人信用等级，并根据对乙方个人信息的评审结果，决定是否审核通过并将乙方的借款需求向甲方进行推荐。

3.3.2 甲方同意向乙方出借相应款项时，已委托丙方在本协议成立后将该笔借款直接划付至乙方账户。甲方授权并委托丙方代其收取本协议所约定的出借人每月应收本息，代收后按照甲方的要求进行处置，乙方对此表示认可。

3.3.3 丙方有权就为本协议借款所提供的服务向甲、乙方收取借款管理费和/或服务费。

3.3.4 甲乙双方同意，从保护甲方合法权益出发，在必要情况下，丙方可以 代甲方向乙方、丁方进行贷款的违约提醒、催收、追索等工作，包括但不限于电话通知、上门通知、发律师函等。该等服务系丙方无偿提供，不得视为丙方负有该义务，或者对甲方承担任何法律责任。乙方及丁方对前述委托的提醒、催收事项已明确知晓并应积极配合，同时应承担因乙方逾期还款及丁方逾期承担保证责任而产生的催收费用及各项违约责任。

3.3.5 丙方接受甲乙双方的委托行为所产生的法律后果由相应委托方承担。如因甲方、乙方、丁方或其他方（包括但不限于技术问题）造成的延误或错误，丙方不承担任何责任。

3.3.6 丙方应对甲方和乙方及丁方的信息及本协议内容保密；如任何一方违约，或因相关权力部门要求（包括但不限于法院、仲裁机构、金融监管机构等），丙方有权披露。

3.4丁方的权利和义务：

3.4.1丁方根据乙方提供的资信材料进行审核评估，确信乙方具有偿还能力，愿意为其在本协议项下的债务提供不可撤销连带担保责任。

3.4.2 如乙方逾期偿还甲方或丙方的借款本息、借款服务费及其他相关费用，则丁方承诺在乙方逾期还款超过 5 日内立即代乙方向甲方及丙方清偿全部的借款本息、罚息或滞纳金、借款服务费及其他各项费用，上述其他费用包括但不限于催收费用、违约金、损害赔偿金及实现债权的费用（包括诉讼费及律师费等）。

3.4.3 保证期间自本协议签署之日起至借款期限届满之日后两年。如乙方履行完毕偿还借款本息、罚息以及与借款相关的各项费用的义务，则丁方的保证责任随即消灭。

3.4.4 丁方代乙方偿还借款本息及其他各项费用后，有权就代为垫付的款项向乙方追偿。

3.4.5 就丁方为乙方提供不可撤销连带保证事宜，乙方同意向丁方提供相应的反担保措施及支付相应的担保服务费，具体反担保措施及担保服务费由乙方和丁方另行协商约定，并书面告知丙方。

3.4.6 丁方确认在保证期间内，甲方可以根据自己的意愿对其根据本协议形成的对乙方的部分或全部债权进行转让，并特此同意甲方届时转让债权无需另行通知丁方且丁方需继续承担保证义务。丁方不得以未接到债权转让通知为由拒绝履行保证义务。

3.4.7 丁方同意并确认：本协议如涉及两个以上的保证人提供连带担保责任，则任一保证人均应履行本合同项下的保证义务，对全部借款本息、罚息、滞纳金、借款服务费、违约金、损害赔偿金、主张债权的费用等承担连带保证责任。甲方和丙方有权向任一保证人主张保证责任。

4. 借款服务费等费用

4.1 在本协议中，“借款管理费”和“居间服务费”是指因丙方为乙方提供信用咨询、评估、还款提醒、账户管理、还款特殊情况沟通等相关服务（统称“信用服务”）而由乙方支付给丙方的报酬（统称借款服务费）。具体金额计算方式以左右逢园网站公告的《左右逢园网站费用及其他规则》为准。

4.2如乙方和丙方协商一致调整借款服务费时，无需经过甲方和丁方同意，且丁方同意就调整后的借款服务费，且仍承担连带担保责任。

5.平台服务费

在本协议中，“平台服务费”是指因丙方为甲方为提供信息交互、撮合、资 信评估等中介服务，而由甲方支付给丙方的报酬（统称平台服务费），具体金额计算方式以左右逢园网站公告的《左右逢园网站费用及其他规则》为准。

6. 违约责任

6.1 合同各方均应严格履行合同义务，非经各方协商一致或依照本协议约定，任何一方不得解除本协议。

6.2 任何一方违约，违约方应承担因违约使得其他各方产生的费用和损失，包括但不限于调查、诉讼费、律师费等，应由违约方承担。如违约方为乙方的或丁方的，则甲方有权立即解除本协议，并要求乙方立即偿还未偿还的本金、利息、罚息，并要求丁方承担连带保证责任。

6.3 本协议提前解除时，若乙方在左右逢园网站的账户里有任何资金余款，丙方有权按照本协议第5.4条的清偿顺序将乙方的余款用于清偿，并要求乙方支付因此产生的相关费用。

6.4 乙方的每期还款均应按照如下顺序清偿：

（1）根据本协议产生的其他全部费用；

（2）罚息；

（3）逾期管理费；

（4）拖欠的利息；

（5）拖欠的本金；

（6）正常的利息；

（7）正常的本金。

6.5 乙方应严格履行还款义务，如乙方逾期还款，则应按照下述条款向甲方支付逾期罚息，自逾期开始之后，逾期本金的正常利息停止计算。

罚息总额 = 逾期本息总额×对应罚息利率×逾期天数
逾期天数 1— 10天 10天及以上
罚息利率 0.05% 0.1%

6.6 乙方应严格履行还款义务，如乙方逾期还款时，则应按照下述条款向丙方支付逾期管理费。

逾期管理费总额 = 逾期本息总额×对应逾期管理费率×逾期天数
逾期天数 1— 10天 10天及以上
逾期管理费费率 0.1% 0.5%

6.7 如果乙方逾期支付任何一期还款超过10天且丁方未承担保证责任代乙方偿还拖欠的借款本息及其他费用的，或连续逾期三期以上（含三期），或累计逾期达五期以上（含五期），或甲方/丙方发现乙方出现逃避、拒绝沟通或拒绝承认欠款事实、故意转让资金、信用情况恶化等危害本协议借款的情形，则本协议项下的全部借款本息提前到期，同时：

（1）乙方应立即清偿本协议下尚未偿付的全部本金、利息、罚息及根据本协议产生的其他全部费用；

（2）丙方有权将乙方的“逾期记录”、“恶意行为”或“不良情况”记入公民征信系统；

（3）丙方有权将乙方违约失信的相关信息及乙方其他信息向包括但不限于媒体、用人单位、公安机关、司法机关及有关逾期款项催收服务机构披露。对此丙方不承担任何责任。

（4）丁方应立即承担连带保证责任，代乙方清偿全部的借款本息、罚息及其他各项费用。

6.8 在乙方还清全部本金、利息、罚息、逾期管理费之前，罚息及逾期管理费的计算不停止。

6.9 本借款协议中的所有甲方与乙方之间的借款均是相互独立的，一旦乙方逾期未归还借款本息，则甲方中的任何一方均有权单独向乙方及丁方追索或者提起诉讼。如乙方逾期支付借款服务费或提供虚假信息的，丙方亦可单独向乙方及丁方追索或者提起诉讼。

6.10 如丁方违反本协议的约定拒绝或逾期承担保证责任的，则丁方除应赔偿甲方由此遭受的实际经济损失外，还应向丙方支付本协议项下的借款金额10%的违约金。

7. 提前还款和提前到期

7.1 乙方可在借款期间任何时候提前偿还全部剩余借款。乙方提前清偿全部剩余借款时，利息计算至乙方清偿全部借款本金之日起的第6日，即应向甲方加付5天的借款利息。

7.2 若乙方在借款期间，出现产品滞销、回款困难、产能下降、非正常停工、市场变化等可能影响乙方还款能力的任何主、客观情形，视为本协议第6.7条约定的“乙方信用情况恶化”。甲方/丙方有权立即向乙方行使不安抗辩权，中止履行本协议，如有未出借给乙方的款项，则暂停出借。乙方收到甲方/丙方的通知后，应在三日内提供金钱或实物担保，确保恢复履行能力。逾期未提供担保的，丙方有权代表甲方通知乙方立即解除本协议，借款提前到期，乙方应当按照本协议第6.7条约定提前还款。

8. 法律适用及争议解决

8.1 本协议的签订、履行、终止、解释均适用中华人民共和国法律。

8.2 本协议在履行过程中，如发生任何争执或纠纷，双方应友好协商解决；若协商不成，任何一方均有权向丙方所在地的人民法院提起诉讼。

9.4 如果本协议中的任何一条或多条违反适用的法律法规，则该条将被视为无效，但该无效条款并不影响本协议其他条款的效力。
