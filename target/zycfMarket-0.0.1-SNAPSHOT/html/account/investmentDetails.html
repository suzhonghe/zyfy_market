    • 常见问题

    • 项目审核

      左右逢园平台积极与地方政府机构和产业园区进行深度合作，为当地优秀实体企业提供融资服务。平台所有借款项目需经过资质审核、实地核查、第三方担保、风控委员会评议等多项严格审核，确认借款人具备还款能力及还款意愿后才能在平台上进行融资。

    • 项目担保

      左右逢园平台上的融资项目均由实力雄厚的第三方融资性担保公司进行担保，同时设置反担保措施，以保证融资项目的还款安全。第三方融资性担保公司对所担保项目提供连带责任担保，如遇项目还款逾期，由担保公司提供全额本息垫付。

    • 信息披露

      左右逢园对所发布所有融资信息进行必要的信息披露，对借款项目的相关资料、资金用途等内容进行在线展示，以保障投资人的知情权，让每一笔投资都做到心中有数。

    • 电子合同

      所投资项目结标后，系统将自动生成等同于纸质合同法律效力的电子合同，以确立借款人与投资人之间的借贷关系。电子合同中包含投资金额、借款利率、还款日期、借贷双方以及平台和担保公司的权利、责任和义务等内容，以保障投资人的利益受到法律保护。

    • 账户安全

      左右逢园平台与第三方资金托管平台进行合作，确保了平台不设置资金池，不触及用户资金。所有用户资金均存放在具有银行托管属性的第三方资金托管平台中，所有资金流转也均通过第三方托管平台进行。

    • 隐私保护

      左右逢园采用权威的VeriSign SSL 256位个人信息加密保护技术进行投资人信息保护，确保投资人提交的个人信息的安全。同时左右逢园对用户资料进行严格的保密，保证不在任何场合和任何时间擅自披露、删除或修改投资人信息。任何申请在左右逢园平台上进行的敏感资料变更，左右逢园都将对申请者进行严格的审核，保障用户账户的信息安全和资金安全。

    1、投资人需要具备什么样的条件？

    年满18周岁，有通过合法途径获得且可自由支配的资金， 建议在不影响生活的情况下合理投资。

    2、导致投标失败的原因是什么？

    投标失败可能源于以下几种原因：
    1.投资人账户内资金不足，需要充值；
    2.网页未及时刷新，此项目投标进度已达到满标状态；
    3.投标时间已截止；
    4.流标。

    3、借款人逾期了怎么办？

    如果借款人发生逾期，为借款项目提供担保的担保公司将在24小时内进行本息垫付。在担保公司无力偿还的情况下，将启用合作担保公司保证金进行垫付，保障投资人的资金安全。

    4、投资后什么时候开始计算收益？

    在借款资金划转到融资方账户的第一个工作日起开始计算收益，节假日顺延。

    5、回款日资金何时到账？

    投资人的利息或本金在回款日的24点前到账均为正常还款。受借款人还款时间、第三方托管平台的业务办理时间等因素的影响，具体还款时间会有所不同。

    6、投资者的收益如何保障？

    作为信息中介平台，主要职能是交易撮合，同时设置多项风险防控措施，对投资者的收益进行保障。
    （1）电子合同，项目结标后平台会产生具有法律效力的《借款协议》，明确投融资双方的债权债务关系；
    （2）第三方担保，实力担保机构提供连带责任担保，逾期由担保公司提供全额本息垫付；
    （3）其它担保措施，为保证资金安全，每个项目都会设置多项风险防控措施，具体参见“项目信息”中的“风险措施”项。
