公司介绍

中阳财富投资控股有限公司是一家经国家工商总局核准的综合性投融资管理机构，于2014年1月在北京成立，注册资本金6500万元。公司自成立以来，始终秉承“产融结合，实业报国”的企业精神，积极响应政府提出的“金融创新服务于实体经济发展”的号召，以支持地方实体经济发展为使命，以产业、科技和金融的高效融合为发展主线，致力于打造符合现代实体经济发展需要的多元化的金融模式。经过多年发展，公司业务涉足网络借贷、股权投资、资产重组与并购等多个领域。

为适应现代金融行业的快速发展，中阳财富不断提升自身在金融创新方面的业务能力。公司充分利用自身在实体产业中的资源优势，以及在金融风控方面的专业优势，与地方实体产业进行深度融合，有效地支持了地方实体经济的健康、快速发展。同时公司利用网络技术，将民间金融与互联网进行有效结合，为广大投融资者搭建了一个全新的投融资信息中介平台——左右逢园（www.zuoyoufy.com），为广大投资者和融资者提供安全可靠、服务专业、项目透明的投融资信息交互服务。

左右逢园介绍

1、左右逢园平台简介

左右逢园（www.zuoyoufy.com）是中阳财富投资控股有限公司全力打造的一家业务规范、管理严格的互联网金融信息中介平台。自上线以来，左右逢园始终积极贯彻执行国家相关政策，以互联网为主要渠道，为借贷双方实现直接借贷提供信息搜集、信息公布、资信评估、信息交互、借贷撮合等服务。

作为一家合法合规的互联网金融服务平台，左右逢园积极与地方实体经济相互融合与发展。平台凭借对实体产业的深刻了解，不断优化业务结构，提升业务能力，甄选和扶持地方优质实体中小企业，努力服务于地方实体经济发展。

与此同时，左右逢园依托自身强大的金融风险管控能力，为投资人筛选和推荐优质投资项目。在确保每一个项目的真实可靠、合法合规的同时，左右逢园努力让投资人能够清晰获知借款方信息、项目资金用途、项目风险程度等重要信息，为投资人的每一次投资行为做好风险防范。

2、收益来源

左右逢园互联网金融服务平台通过与地方政府及国内知名产业园进行战略合作，在为优秀实体企业提供融资服务的同时，为投资者筛选和推荐优质投资项目。融资企业通过平台获得投资人出借的资金，并将其用于日常经营。左右逢园通过一系列风控措施保障项目的真实性和可靠性，并全程监控项目的还款情况。项目到期后，投资人收回本金，并与企业共同分享由企业经营产生的收益。
